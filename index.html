TENTANG SIMPUS

SIMPUS adalah singkatan dari Sistem Informasi Manajemen Puskesmas. Fitur SIMPUS antara lain.

Mudah Digunakan

Aplikasi SIMPUS menggunakan aplikasi web. Hanya dengan sebuah browser, kita sudah dapat menjalankannya.

Keyboard Support

Aplikasi SIMPUS mengadaptasi kemudahan aplikasi desktop.

Data Tersentralisasi

Server terpusat, sehingga tidak perlu khawatir mengenai pemeliharaan aplikasi.

Desain Responsif

Desain responsif membuat aplikasi dapat diakses dengan layar Smartphone maupun Tablet.

ICD-X

Diagnosa yang digunakan sudah menggunakan standar WHO.

Integrasi BPJS

Memudahkan entri ke dalam BPJS (Single Entry)

Gudang Farmasi

Integrasi SIMPUS dengan Software Gudang Farmasi

Software Antrian

Integrasi SIMPUS dengan Software Display Antrian

Fitur pada Aplikasi SIMPUS

  • Hak Akses User seperti Super Admin, Administrator Puskesmas, Petugas Puskesmas (Tergantung Ruangannya), Dinas Kesehatan, Petugas Luar Gedung (Pustu, Polindes, Ponkesdes, Posyandu).
  • Loket Pendaftaran
  • Cetak Kartu Pasien
  • Rekam Medik (Rawat Jalan & Rawat Inap)
  • Klinik / Poli (Umum, Gigi, KIA, Lansia, Anak, Gizi, IVA, VCT, UGD, Rawat Inap, PONED)
  • Master Data (Pasien, Obat, Injeksi, BHP, Tindakan, Laborat, Departemen, Wilayah Kerja, Pegawai)
  • Manajemen Obat (Gudang Obat, Apotek, Stock Obat, Kartu Stok)
  • Laporan (Meliputi semua kebutuhan baik harian maupun bulanan)
  • Edit Data Kunjungan
  • Pengaturan Puskesmas
  • Laboratorium
  • Kasir
  • Menu yang dapat di-kustomisasi
  • Reset Data Transaksi
  • Log User
  • Summary Chart
  • Keyboard Support

Fitur pada Aplikasi Antrian

  • Mendukung Sistem Operasi Windows, Mac OS, dan Linux.
  • Web based system untuk manajemen konten.
  • Cetak Antrian.
  • Pemanggilan nomor antrian.
  • Dapat diintegrasikan dengan simpus.
  • Jaringan menggunakan Local Intranet.
  • Dapat disisipkan video Promosi Kesehatan.
